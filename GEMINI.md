# Gemini AI Instruction Manual

## Tech Stack

This project uses the following technologies:
-   **Language:** TypeScript
-   **Framework:** React
-   **Build Tool:** Vite
-   **Styling:** Styled Components
-   **Animation:** GSAP
-   **Routing:** React Router DOM
-   **Utilities:** Lodash

## Coding Style

-   We use `PascalCase` for React components.
-   We use `camelCase` for variables and functions.
-   We prefer functional components with hooks over class-based components.

## Forbidden Actions

-   Never delete the user database.
-   Never use `eval()`.
-   Do not introduce any new dependencies without prior approval.

## Project Goal

This project is a personal portfolio website for Benjamin. It showcases his projects, skills, and provides contact information.
